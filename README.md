# Plant Pathogen Classification using CNN
Built a convolutional neural network using TensorFlow/Keras to classify plant pathogens from a 40,000-image dataset as part of a research project at Miami Dade College. Evaluated model performance using a confusion matrix and analyzed precision, recall, and accuracy to assess classification reliability across 7 classes.
This project marked my transition from an initial interest in neuroscience to a deeper focus on data science and machine learning. While building a convolutional neural network using Python and TensorFlow, I gained hands-on experience with image classification, data preprocessing, and model evaluation. Working through this project helped me understand the practical power of artificial intelligence in scientific research and its applications in real-world problem solving.
It strengthened my interest in machine learning and showed me how programming can be used to extract meaningful insights from complex data. I am now motivated to continue developing projects in AI and pursue internships where I can further improve my technical skills and contribute to real-world machine learning applications.
